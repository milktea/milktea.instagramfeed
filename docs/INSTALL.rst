milktea.instagramfeed Installation
----------------------------------

To install milktea.instagramfeed using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``milktea.instagramfeed`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        milktea.instagramfeed

* Re-run buildout, e.g. with:

    $ ./bin/buildout
